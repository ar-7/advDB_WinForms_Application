To import this program for marking:

- Create a new database in mySQL called hotel_booking

- Import hotel_booking.sql

- Change mySQL root password to: mysqlroot

- This link contains information on how to access mySQL after changing root password: https://artisansweb.net/set-mysql-root-password-phpmyadmin/

- Open Visual Studio project using ROB19002438_LP20549A1_advDB.sln

- Use the server explorer to make sure the connection is active.

- Visual studio will likely require mySQL connector for NET and mySQL for visual studio, available at the links below:

https://dev.mysql.com/downloads/windows/visualstudio/

https://dev.mysql.com/downloads/connector/net/
